# Data & code — NCOA1 NR1 gBGC

A curated, self-contained package of the data, analysis outputs, and figure-generating
code for:

> *Collateral loss of a conserved nuclear-receptor coactivator motif by GC-biased gene
> conversion in monotremes.*

It contains everything needed to reproduce every figure and every numeric value reported
in the paper. It deliberately omits unrelated pipeline data and other projects.

## Contents

```
analysis/
  outgroups_48sp/      48-species NCOA1 codon + protein alignments, ML tree
  runs_48sp/           HyPhy selection JSONs (aBSREL, BUSTED, Contrast-FEL, FUBAR),
                       codeml ancestral reconstruction (asr/rst), labeled trees
  gbgc_genomic/        per-species NCOA1 genomic loci (~26 kb windows) + analysis scripts
  coactivator_scan/    nine-coactivator LXXLL scan inputs + output table

figures/
  Fig{1-4}_gbgc.*           main figures (pdf/png/svg) as generated by the build scripts
  FigS_{codonpos,cpgisland}_gbgc.*   supplementary figures
  build_gbgc_*.py           figure-generating scripts (data → figure)

submission/supp_tables/  Supplementary Tables S1–S4 (.tsv/.xlsx)
```

## Reproducing the results

All scripts anchor their paths relative to this paper directory; run them from anywhere.

```bash
# Every prose-reported value (GC/percentile/CpG o/e; composition; W→S counts +
# Fisher's exact; NR1 site-level FUBAR/Contrast-FEL; platypus–echidna identities):
python analysis/gbgc_genomic/compute_reported_values.py

# Regenerate each figure from the committed data:
python figures/build_gbgc_fig1.py        # motif + ancestral reconstruction
python figures/build_gbgc_fig2.py        # GC by clade + genomic tract
python figures/build_gbgc_fig3.py        # W→S bias + platypus–echidna retention
python figures/build_gbgc_fig4.py        # coactivator LXXLL scan
python figures/build_gbgc_figS_codonpos.py
python figures/build_gbgc_figS_cpgisland.py

# Re-fetch the raw genomic loci from NCBI (set your email first; optional):
export ENTREZ_EMAIL="you@example.com"
python analysis/gbgc_genomic/fetch_reference_genomic.py
python analysis/gbgc_genomic/validate_genomic_fastas.py   # QC: each window contains the NR1 exon
```

## Raw data

Source coding/genomic sequences are the public NCBI / Ensembl accessions listed in
Supplementary Table S2 (per-species accessions). The genomic-locus FASTAs in
`analysis/gbgc_genomic/` are ~26-kb windows fetched from those accessions by
`fetch_reference_genomic.py`.

## Generated figures

`figures/Fig*_gbgc.*` are produced directly by the `build_gbgc_*.py` scripts — they are the
machine-generated, data-faithful versions (pdf/png/svg).

## Dependencies

Python 3 with `biopython`, `numpy`, `scipy`, `matplotlib`; HyPhy and PAML/codeml are needed
only to regenerate the selection JSONs / `rst` from scratch (the committed outputs let you
reproduce all figures without them).
